error_t route ( input_device_t  in,
output_device_t  out 
)

Routes an analog input directly to an analog output, bypassing the ADC/DAC (e.g. for an analog monitoring/passthrough path). Only ADC_INPUT_LINE1 and ADC_INPUT_LINE2 are supported as input.

Parameters
inADC_INPUT_LINE1 or ADC_INPUT_LINE2
outDAC_OUTPUT_NONE, DAC_OUTPUT_LINE1, DAC_OUTPUT_LINE2 or DAC_OUTPUT_ALL see https://m5stack-doc.oss-cn-shenzhen.aliyuncs.com/1141/ES8388.pdf

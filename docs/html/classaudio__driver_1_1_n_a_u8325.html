Detailed Description

Header-only driver for the Nuvoton NAU8325 Class-D amplifier with DAC.

The NAU8325 is an I2C-controlled mono Class-D amplifier with an integrated 16/20/24/32-bit DAC. This driver configures the clock tree (MCLK → N1/N2/N3 dividers → DAC clock), I2S format, oversampling rate, ALC, and analog bias registers. All configuration state is held per-instance, so multiple devices on the same bus are supported.

Typical usage:

NAU8325 amp(i2c_handle);
amp.begin(44100, 16, 256); // 44.1 kHz, 16-bit, MCLK = 256 × FS
amp.setVolume(0xC0, 0xC0); // ~75 % volume on both channels

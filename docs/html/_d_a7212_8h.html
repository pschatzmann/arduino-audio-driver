Detailed Description

Header only C++ driver for the Dialog Semiconductor DA7212 audio codec.

Ported from the Zephyr RTOS driver (drivers/audio/da7212.c / da7212.h, Copyright 2025 NXP, Apache-2.0).

Note
The PLL is not configured by this driver: it keeps the PLL disabled and uses MCLK directly as the system clock (matching the Zephyr driver's default, devicetree independent, configuration path).
